TEST NO: 13

TITLE: Animation Deformation Display Modes - Relative vs Absolute

DESCRIPTION: Validates that the "Show Absolute Deformations" checkbox correctly controls how deformed mesh coordinates are displayed during animation playback, allowing users to choose between relative motion visualization and absolute deformation display

INPUTS:

- Modal coordinates, stresses, and deformations all loaded successfully
- Solver completed with Von-Mises stress output
- Animation time range: 0.3s to 0.8s (intentionally not starting at t=0 to show difference)
- Deformation scale factor: 2.0
- Custom time step mode with 0.05s intervals

TEST STEPS:

1. Complete solver run with Von-Mises stress and Include Deformations enabled
2. Switch to Display tab after solver completion
3. Locate "Show Absolute Deformations" checkbox in Visualization Controls group
4. Verify checkbox is visible and unchecked by default (Relative mode)
5. Set animation time range: Start = 0.3 seconds, End = 0.8 seconds
6. Set time step mode to "Custom Time Step" with step = 0.05 seconds
7. Set deformation scale factor to 2.0
8. Click Play to start animation in Relative mode (checkbox unchecked)
9. Observe and note that first animation frame appears at undeformed mesh position
10. Stop animation after observing a few frames
11. CHECK the "Show Absolute Deformations" checkbox (switch to Absolute mode)
12. Click Play again to start animation in Absolute mode
13. Observe and note that first animation frame shows deformed position
14. Verify stress scalar values displayed are identical in both modes
15. Verify time text overlay updates correctly in both modes
16. Stop animation

EXPECTED RESULTS:

Relative Mode (Checkbox Unchecked - Default):
- Checkbox is unchecked when first visible
- First animation frame (t=0.3s) appears at original undeformed mesh position
- Animation shows motion relative to t=0.3s start time
- Useful for visualizing motion patterns and dynamics
- Helps highlight oscillations and vibrations

Absolute Mode (Checkbox Checked):
- First animation frame (t=0.3s) shows true deformed position
- Includes any pre-existing deformation accumulated from t=0 to t=0.3s
- Shows absolute displacement values throughout animation
- Better for quantitative analysis

Both Modes:
- Stress scalar values (Von-Mises) are identical - not affected by checkbox
- Time text overlay updates correctly showing current time
- Animation playback is smooth in both modes
- Deformation scale factor (2.0) applies equally in both modes
- Pause, Play, and Stop controls work identically
- Tooltip on checkbox explains both modes clearly
